Specifies wheter the control is dragging immediatly when the mouse is down.

 

__property bool AutomaticMove = { read=FAutomaticMove, write=SetAutomaticMove };

 

Description
Use AutomaticMove to specify whether the control is dragging automatically. If AutomaticMove is true, a drag operation is immediatly performed when the mouse is down.
 

 

Indicates whether the MainForm is transparent in according to TransparentColor property.


__property bool TransparentMainForm = { read=FTransparentMainForm, write=SetTransparentMainForm };

 

Description

Use TransparentForm to specify that the MainForm be drawn transparently. If TransparentForm is true, the MainForm is transparent.

 

Note: This property work with Property of the MainForm ( BorderStyle, Color, TransparentColor, TransparentColorValue ).         

 

Specifies the Bitmap that appears on the Canvas of the control.


__property Graphics::TBitmap* Bitmap = { read=FBitmap, write=SetBitmap };

 

Description

Use Bitmap to load a bitmap wich will showing on the Canvas of the component.

 

 

Indicates whether the bitmap covers its rectangular area.
 

__property bool Transparent = { read=FTransparent, write=SetTransparent };
 

Description

Use Transparent to specify that the bitmap be drawn transparently. If Transparent is true the transparent area is clipping, in according to TransparentColor property. If Transparent is false, this area is recreated.

Note: This property work with "Region functions" of Win SDK.

 

Determines which color of the bitmap is to be transparent when the bitmap is drawn.

 

__property TColor TransparentColor = { read=FTransparentColor, write=SetTransparentColor };

 

Description
Use TransparentColor to determine how to draw the bitmap transparently.

 

Determines the range of the transparent colors, when the Transparent property is true.


__property int TransparentTolerance = { read=FTransparentTolerance, write=SetTransparentTolerance };
 

Description

Use TransparentTolerance to specify a range of transparent colors. A value of 0 make transparent only the pixels of the Bitmap property which have a color equal to TransparentColor property. With a value x>0, will be transparent all the pixels in a range of color (TransparentColor-x, TransparentColor+x).

 

Determines the degree of translucence of the MainForm.

 

__property int Translucence = { read=FTranslucence, write=SetTranslucence };
 

Description

Use translucence for make semi-invisible the MainForm of the application. This work with AlphaBlend and AlphaBlendValue of  MainForm.

 

 

For all others properties, see the standard documentation of TCustomControl and Ancestors.
