Specifies wheter the control size is setted automatically.

 

__property bool AutoSize = { read=FAutoSize, write=SetAutoSize };

 

Description
Use AutoSize to specify whether the control size is setted automatically. If AutoSize is true, the control is resized automatically in accord to Bitmap size.

Determines the behaviour of the focus rectangle, showing on control.


          __property TFocusMode FocusMode = { read=FFocusMode, write=SetFocusMode };
 

Description:

Use FocusMode to change the behaviour of the focus rectangle. If FocusMode is fmFixed, the focus rectangle is persistent. If the FocusMode is fmAutomatic, the FocusRect is not show initially. Only when tab-key is down, the focus rectangle is showing. After 5 seconds it become invisible.

 

 

Determines the Indent of the focus rectangle showing.

 

          __property int FocusRectIndent = { read=FFocusRectIndent, write=SetFocusRectIndent };

 

Description

Use FocusRectIndent to change the size of the focus rectangle. If the FocusRectIndent value is invalid, the value is calculating automatically.

 

Determines the amount of time during witch the focus rectangle is visible, when the FocusMode property is fmAutomatic.

 

          __property int FocusTime = { read=FFocusRectIndent, write=SetFocusRectIndent };

 

 

The control get the bitmap from a TImage object.

 

          __property TImage Image = { read=FImage, write=SetImage };

 

It is an alternative to the Bitmap property. Set the Image property to a TImage object for assign the same image to more SkinButtons, using an only image.

 

 

 

Determines the number of image on the Bitmap property. See the Bitmap property for more details.

 

          __property int NumImage = { read=FNumImage, write=SetNumImage };

 

The min value is 1, and the max value is 5.

 

 

Indicates the visibility of the focus rectangle.

 

          __property bool ShowFocusRect = { read=FShowFocusRect, write=SetShowFocusRect };
 

Description

Use ShowFocusRect to manage the visibility of the focus rectangle. If ShowFocusRect is true, the focus rectangle is visible, also it is invisible.

 

 

Indicates whether the bitmap covers its rectangular area.
 

          __property bool Transparent = { read=FTransparent, write=SetTransparent };
 

Description

Use Transparent to specify that the bitmap be drawn transparently. If Transparent is true the transparent area is clipping, in according to TransparentColor property. If Transparent is false, this area is recreated.

Note: This property work with "Region functions" of Win SDK.

 

 

Determines which color of the bitmap is to be transparent when the bitmap is drawn.

 

          __property TColor TransparentColor = { read=FTransparentColor, write=SetTransparentColor };

 

Description
Use TransparentColor to determine how to draw the bitmap transparently.

 

 

Determines the behaviour of the Transparent area of the control.

 

          __property TTransparentMode TransparentMode = { read=FTransparentMode, write=SetTransparentMode };

 

Description

If TransparentMode is tmOperative, the transparent area is operative and respond to mouse actions.

 

 

If TransparentMode is tmCutting the transparent area above is not operative, thus it not respond to mouse input.

 

 

Determines the range of the transparent colors, when the Transparent property is true.


          __property int TransparentTolerance = { read=FTransparentTolerance, write=SetTransparentTolerance };
 

Description

Use TransparentTolerance to specify a range of transparent colors. A value of 0 make transparent only the pixels of the Bitmap property which have a color equal to TransparentColor property. With a value x>0, will be transparent all the pixels in a range of color (TransparentColor-x, TransparentColor+x).

 

 

For all others properties, see the standard documentation of TCustomControl and Ancestors.
