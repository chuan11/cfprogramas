Occurs when the mouse pointer moves over the control.

  

       __property TNotifyEvent OnMouseEnter = { read=FOnMouseEnter, write=FOnMouseEnter };

 

Description
Write an OnMouseEnter event handler to take specific action when the user moves the mouse over the SkinButton control. For example, you can use this event to change the font color when the mouse is over the control, and then use the OnMouseLeave event to change it back when the mouse moves off the control.

 

 

Occurs when the mouse pointer moves off from over the control.

     

          __property TNotifyEvent OnMouseLeave = { read=FOnMouseLeave, write=FOnMouseLeave };
 

Description
Write an OnMouseLeave event handler to take specific action when the user moves the mouse off the SkinButton control. For example, you can use this event to undo changes that were made in an OnMouseEnter event handler.
