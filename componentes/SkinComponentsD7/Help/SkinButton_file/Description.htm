Description

Use TSkinButton to put on a windowed control, an advanced graphic control for designing and customizing graphic buttons.

Advanced features for a complete control:

     

     Derive the graphic aspect of the control from a bitmap.

     Manage the transparence of the control.

     Manage the transparent color of the control.

     Manage the tolerance of the area transparent.     

     Manage the color of the control.

     Manage the behaviour of the focus rectangle ( Automatic or fixed ).

     Manage the visibility of the focus rectangle.

     Manage the Indent of the focus rectangle.

     

     and more…
