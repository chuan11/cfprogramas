Creates and initializes an instance of TSkinForm.

 

__fastcall TSkinForm(TComponent* Owner);
 

Description
Call TSkinForm indirectly, using the new keyword, to instantiate a TSkinForm object at runtime. SkinForm placed on forms at design time are created automatically.

AOwner is the component that is responsible for freeing the TSkinForm instance. It becomes the value of the Owner property.

 

 

Destroys an instance of TSkinForm.

 

__fastcall ~TSkinForm(void);

 

Description

Do not call ~TSkinForm directly in an application. Instead, use the delete keyword.

As the control is destroyed, it frees the TBitmap object in the Bitmap property.

 

 

For all others methods, see the standard documentation of TCustomControl and Ancestors.
