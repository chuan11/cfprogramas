 Overview :
 ----------
 The ** TmxOneInstance ** is a component to allow to 
 start only one copy of your application.

 Help file is not available at the moment. 
 I am sorry! I hope the demo application helps you.
 
*******************************************************
 
 How to install :
 ----------------

 Delphi 5:
 
 Open mxOneInstance_d5.dpk project file, compile and 
 install it.
 Now TmxOneInstance will  be available in Max Palette.

 Delphi 6:
 
 Open mxOneInstance_d6.dpk project file, compile and 
 install it.
 Now TmxOneInstance will  be available in Max Palette.

 Delphi 7:
 
 Open mxOneInstance_d7.dpk project file, compile and 
 install it.
 Now TmxOneInstance will  be available in Max Palette.

 Delphi 2005:
 
 Open mxOneInstance_d9.dpk project file, compile and 
 install it.
 Now TmxOneInstance will  be available in Max Palette.

*******************************************************

 How to use:
 -----------

 Just dropo this component on your application's main 
 form.
